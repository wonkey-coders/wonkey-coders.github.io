##### Method RemoveLast:T(  )
Removes the last value in the list equal to a given value.

Removes and returns the last value in the list.

In debug builds, a runtime error will occur if the list is empty.

| Parameters |    |
|:-----------|:---|
| `value` | value The value to remove. |
